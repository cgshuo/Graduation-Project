Forms-XML is an interactive component (ActiveX control) invoked by applications for generating user interfaces for prescrib ed XML vocabularies automatically [1,2,3]. Based on a given XML schema, the component generates a hierarchy of HTML forms for users to interact with and update XML data compliant with the given schema. The user interface Forms-XML generates is very simple with an abundance of guidance and developers. Forms-XML Designer. It supports interface designers to specify the layout and style of interface can be generated immediately by invoking the Forms-XML component. We will demonstrate the functions of Forms-XML by demonstrating the Designer. The form-based user interface Forms-XML generates and the process for customizing a user interface for an XML vocabulary will be exhibited by examples.
