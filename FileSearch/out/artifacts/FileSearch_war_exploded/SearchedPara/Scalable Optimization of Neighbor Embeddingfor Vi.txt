
Brief description of the datasets  X  Iris: the UCI Iris dataset.  X  ORL: the AT&amp;T ORL database of face images, each image of size 92  X  112  X  COIL: the COIL-20 dataset from Columbia University Image Library, toy images of different angles, each image of size 128  X  128.  X  Seg: the UCI Image Segmentation dataset, image patches from 7 outdoor images, originally with 19 high-level features.  X  WebKB: the WebKB4 dataset from CMU Text Learning group, text doc-uments; 10,000 words with maximum information gain are preserved.  X  7Sectors: the 4 Universities dataset from CMU Text Learning group, text documents classified to 7 sectors; 10,000 words with maximum information gain are preserved.  X  OptDig: the UCI optical recognition of handwritten digits , originally with 64 dimensions.  X  Reuters: the UCI Reuters-21578 dataset, text documents, with 18933 words.  X  RCV1: text documents from four classes, with 29992 words.  X  Spam: A database for spam email classification, 448 numerical features for each email.  X  PegDig: the UCI pen-based recognition of handwritten digits dataset, orig-inally with 16 dimensions.  X  Magic: the UCI MAGIC Gamma Telescope Data Set , 11 numerical fea-tures.  X  Shuttle: the UCI Statlog (Shuttle) Data Set , 9 numerical features.
